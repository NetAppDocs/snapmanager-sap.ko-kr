= UNIX 호스트에서 소프트웨어를 제거합니다
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
SnapManager 소프트웨어가 더 이상 필요하지 않으면 호스트 서버에서 제거할 수 있습니다.

.단계
. 루트로 로그인합니다.
. 서버를 중지하려면 '* smsap_server stop*' 명령을 입력합니다
. SnapManager 소프트웨어를 제거하려면 다음 명령을 입력합니다.
+
``UninstallSmsap *’

. 소개 텍스트가 끝나면 * Enter * 를 눌러 계속합니다.
+
제거가 완료됩니다.
