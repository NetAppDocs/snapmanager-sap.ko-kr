= 데이터베이스를 구성합니다
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
SnapManager를 사용하여 백업할 대상 데이터베이스와 대상 데이터베이스 메타데이터를 저장할 저장소 데이터베이스를 각각 최소 두 개 이상의 데이터베이스를 구성해야 합니다. SnapManager 작업을 수행하기 전에 타겟 데이터베이스와 SnapManager 저장소 데이터베이스를 구성하고 온라인 상태로 만들어야 합니다.
