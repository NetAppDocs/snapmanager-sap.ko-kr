= 프로파일에 대한 액세스 권한 부여
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
SnapManager를 사용하면 무단 액세스를 방지하기 위해 프로필에 대한 암호를 설정할 수 있습니다.

.단계
. 다음 명령을 입력합니다.
+
'msap credential set-profile-name profile_name[-password password]'
