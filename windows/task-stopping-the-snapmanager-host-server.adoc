= SnapManager 호스트 서버를 중지하는 중입니다
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
SnapManager 사용을 마치면 서버를 중지할 수 있습니다.

. 서버를 중지하려면 루트 사용자로 다음 명령을 입력합니다.
+
'msap_server stop'입니다
