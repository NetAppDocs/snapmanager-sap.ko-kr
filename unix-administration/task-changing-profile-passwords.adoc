= 프로파일 암호를 변경합니다
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
리포지토리의 기존 프로파일을 보호하려면 프로파일의 암호를 업데이트해야 합니다. 이 프로파일을 사용하여 백업을 생성할 때 이 업데이트된 암호를 적용할 수 있습니다.

.단계
. 기존 프로파일의 프로파일 암호를 업데이트하려면 다음 명령을 입력합니다.
+
'* smssap profile update-profile_profile_name_-profile-_password_*'
